# Гид планировок
Каталог планировок квартир Ростова-на-Дону с ознакомительной информацией о ценах и типах планировок. Проект носит исключительно информационный характер.

## О проекте

Сайт создан для ознакомления пользователей с различными типами планировок квартир и ориентировочными ценами.

На сайте публикуется:

- виды планировок;
- фотографии и схемы;
- описание особенностей;
- ориентировочные цены;
- полезная информация при выборе квартиры.

## Важно

Вся информация представлена исключительно в ознакомительных целях.

Цены могут отличаться от актуальных предложений застройщиков и продавцов.

Проект не является агентством недвижимости и не оказывает услуги по продаже недвижимости.
